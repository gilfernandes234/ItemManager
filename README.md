# ItemManager
Editor for better editing of spr and dat images and files. 

features:

-Image Enhancer
 - Upscale: 2, 4, 8
 - Denoise Level: 1, 2, 3 
 - Resize: 32x32, 64x64, 128x128, 512x512
 - Custom Size

- Image Edit
  - Mirror Vertical
  - Mirror Horizontal
  - Color Adjust
  - Rotation
 
- Spr/Dat Editor
  - Edit Id
  - Flags Adjust
  

Manipulating the dat and spr files is only 
possible for versions 10.98; 
it is not recommended for other versions.

- py -m pip install pillow
- py -m pip install customtkinter
